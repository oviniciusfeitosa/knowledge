# PHPStorm

## Keymap - Clone Caret

* Settings  \(Ctrl + Alt + S\)
* Keymap 
* Find: **`Clone Caret`**
* Define your keymap to Above\(up\) and Below\(down\)

```text
#Example
CTRL + Shift + Alt + Up
CTRL + Shift + Alt + Down
```
